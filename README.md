# Python Todo Manager

A simple command-line Todo Manager built with Python. This project helps users manage daily tasks, track progress, organize priorities, save data, and perform task-related operations through a menu-driven interface.

## Features

* Add new tasks
* View all tasks
* Delete tasks
* Edit existing tasks
* Mark tasks as completed
* Search tasks by keyword
* Repeat tasks multiple times
* Daily schedule viewer
* Save tasks to a JSON file
* Load tasks from a JSON file
* Read and write notes using text files
* Date checking functionality
* Colored task status output using Colorama

## Technologies Used

* Python
* JSON
* File Handling
* Colorama

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/python-todo-manager.git
```

2. Navigate to the project folder:

```bash
cd python-todo-manager
```

3. Install Colorama:

```bash
pip install colorama
```

4. Run the program:

```bash
python main.py
```
